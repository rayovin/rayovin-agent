# 🤖 Rayovin Agent (رایوین)

<div dir="rtl" align="right">

## عامل هوشمند فارسی | محصول شرکت نبض آینده جنوب

**Rayovin Agent** یه عامل هوشمند (Agent) تمام‌عیاره که مثل یه دستیار شخصی همه‌جا همراته — تو تلگرام، دیسکورد، خط فرمان، یا حتی دسکتاپ. از تجربه‌هاش یاد می‌گیره، مهارت (Skill) می‌سازه، حافظه داره، و می‌تونه کارها رو به صورت زمان‌بندی شده یا موازی انجام بده.

---

## ✨ ویژگی‌های کلیدی

### 🧠 یادگیری خودکار
- از مکالماتت Skill می‌سازه و خودش رو بهتر می‌کنه
- حافظه داره — جلسات قبلی رو به خاطر میاره
- دانشش رو بین جلسات مختلف حفظ می‌کنه

### 📱 همه‌جا همراته
- **تلگرام**، **دیسکورد**، **واتساپ**، **سیگنال**، **اسلک**
- **خط فرمان (CLI)** و **دسکتاپ**
- همه از یه gateway واحد

### ⏰ کارهای زمان‌بندی شده
- گزارش روزانه، پشتیبان شبانه، بررسی هفتگی
- تحویل خودکار به هر پلتفرمی

### 🔄 کار موازی
- می‌تونه چند تا زیرعامل (sub-agent) رو همزمان اجرا کنه
- کارهای سنگین رو به چند بخش تقسیم می‌کنه

### 🛠 Skillهای متنوع
- برنامه‌نویسی (PHP، Python، C#، Frontend، Backend و...)
- امنیت، API، DevOps
- تحلیل داده، هوش مصنوعی
- و کلی Skill دیگه

---

## 🚀 نصب سریع

### لینوکس، مک، WSL2

```bash
curl -fsSL https://rayovin.github.io/rayovin-agent/install.sh | bash
```

### نصب با pip

```bash
pip install rayovin-agent
```

### اجرا

```bash
rayovin chat
```

## 🎯 چجوری کار می‌کنه؟

| ویژگی | توضیح |
|-------|--------|
| **CLI/TUI** | محیط ترمینال با autocomplete، history، و ابزارهای متنوع |
| **پیام‌رسان‌ها** | تلگرام، دیسکورد، واتساپ، سیگنال، اسلک و ۲۰+ پلتفرم دیگه |
| **یادگیری** | حافظه، ساخت Skill خودکار، جستجوی جلسات قبل |
| **زمان‌بندی** | Cron داخلی — کارها رو در زمان مشخص انجام بده |
| **زیرعامل** | کارهای سنگین رو به agentهای کوچیکتر بسپار |
| **هر جا اجرا میشه** | سرور $5، کلاستر GPU، داکر، Modal، SSH |
